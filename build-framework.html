Hack on Errai: Build the framework yourself in Eclipse

Get Started

  • Fork the Errai repository on Github
  • Clone your fork
  • Get Eclipse
  • Ensure your Eclipse has m2e
  • File → Import → Existing Maven Projects...
  • Fix up build failures with Build Path → Use as Source Folder or Build Path → Exclude
Community
  • Community forum – get unstuck
  • errai-dev – discuss big changes
  • #errai on freenode.net – chat on IRC
  • Issue Tracker – report & track issues
